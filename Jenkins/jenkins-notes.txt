Install Jenkins a root
----------------------
1) Download jenkins package(Stable LTS) in https://pkg.jenkins.io/redhat-stable/
2) To use this repository, run the following command:
  sudo wget -O /etc/yum.repos.d/jenkins.repo https://pkg.jenkins.io/redhat-stable/jenkins.repo
  sudo rpm --import https://pkg.jenkins.io/redhat-stable/jenkins.io.key
3) If you've previously imported the key from Jenkins, the rpm --import will fail because you already have a key. Please ignore that and move on.
  yum install epel-release # repository that provides 'daemonize'
  yum install java-11-openjdk-devel
  yum install jenkins
4) sudo amazon-linux-extras install epel
5) amazon-linux-extras install java-openjdk11
6) yum install jenkins
7) service jenkins enable | service jenkins start
8) access jenkins web ui and update admin password
   cat /var/lib/jenkins/secrets/initialAdminPassword
9) yum install git
